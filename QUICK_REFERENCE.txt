# Power Mode Toggle - Quick Reference

## Quick Start
1. Right-click `PowerModeToggle.bat`
2. Select **"Run as administrator"**
3. Watch the toggle happen automatically
4. Press any key to close the window

## Common Commands (Reference)

```bash
# Check current active power plan
powercfg /getactivescheme

# List all power plans
powercfg /list

# Switch to a specific power plan (replace GUID)
powercfg /setactive GUID

# Get detailed info about a power plan
powercfg /query GUID
```

## Desktop Shortcut Setup (with Admin)

**Steps:**
1. Right-click desktop → New → Shortcut
2. Enter: `C:\path\to\PowerModeToggle.bat`
3. Name it: "Toggle Power Mode"
4. Right-click shortcut → Properties
5. Click "Advanced"
6. ✓ Check "Run as administrator"
7. Click OK → Apply → OK

Now double-click to toggle (admin prompt may appear)

## What This Script Does

1. ✓ Checks you're running as admin
2. ✓ Detects your current power mode
3. ✓ Enables Ultimate Performance if missing
4. ✓ Switches to the opposite mode
5. ✓ Shows you the result
6. ✓ Keeps log in console

---

## Exit Codes (for advanced users)

- `0` = Success ✓
- `1` = Not Admin ✗
- `2` = Can't enable Ultimate Performance ✗
- `3` = Can't switch modes ✗
- `4` = Unknown power plan ✗
